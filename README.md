# NovaNetwork SMP Website

A static Minecraft server website built for Vercel deployment.

## Deployment Instructions (No Terminal Required)
1. Upload all these files and folders into a new GitHub repository.
2. Ensure your logo is uploaded to `/assets/images/logo.png`.
3. Go to [Vercel](https://vercel.com) and log in with GitHub.
4. Click **Add New -> Project**.
5. Import this GitHub repository.
6. Click **Deploy**. Vercel will handle everything automatically!
